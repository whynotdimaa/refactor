# Restaurant Management System

Система управління рестораном на базі Flask, що забезпечує автоматизацію роботи з меню, замовленнями та столиками[cite: 11, 14].

## Основні можливості
- [cite_start]**Керування меню**: додавання, редагування та видалення страв[cite: 14].
- [cite_start]**Рольова модель**: окремі інтерфейси для адміністратора, офіціанта та кухаря[cite: 14].
- [cite_start]**Замовлення та доставка**: створення замовлень, відстеження статусів та адресна доставка[cite: 14].
- [cite_start]**Система відгуків**: можливість залишати оцінки та коментарі до страв[cite: 14].

## Вимоги та запуск
[cite_start]Для роботи застосунку потрібен Python 3.10+ та встановлені бібліотеки[cite: 213].

### Встановлення залежностей:
```bash
pip install flask flask-sqlalchemy flask-bcrypt flask-login pytest


Запуск сервера:
python refactored_code.py
Запуск тестів:
python -m pytest tests/test_cases.py -v
```
